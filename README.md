DogFood - онлайн магазин товаров для собак

В проекте реализованы разделы:
- Главная страница
- Каталог товаров
- Корзина
- Любимые товары
- Страница аккаунта
- Страница добавления товара
- Страница одного товара

При реализации данного проекта, были использованы технологии:
- React
- JavaScript
- Rest API
- CSS
Библиотеки:
- Bootstrap
- Bootstrap icons
